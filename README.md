# google-maps-example

Google Maps in Elm 0.18!

## Setup

```
npm install -g elm elm-live
npm install -g bower
```

## Installation

```shell
git clone git@github.com:fredcy/elm-polymer-calendar.git
cd elm-polymer-calendar
bower install
make
```

## Serving
```
cd elm-polymer-calendar
python3 -m http.server
```

Browse http://localhost:8000/
